# Social Media Creative Specifications

**Last Updated:** 2025-12-09  
**Version:** 1.0

## Overview

Detailed specifications for creating social media graphics. Use these specs when designing creatives for Facebook, Zalo, Instagram, and other platforms.

## Creative Templates

### Template 1: Feature Highlight

**Layout:**
```
┌─────────────────────────────┐
│ [Icon/Screenshot]           │
│                             │
│ HEADLINE (Bold, 32px)       │
│                             │
│ Benefit statement           │
│ (16px, 2-3 lines)           │
│                             │
│ [CTA Button]                │
│                             │
│ Brand logo (bottom right)   │
└─────────────────────────────┘
```

**Colors:**
- Background: White or Light Gray (#F5F5F5)
- Headline: Deep Navy (#1A2B4A)
- Body text: Secondary Text (#4A4A4A)
- CTA: Teal (#00A8A8)
- Accent: Gold (#D4AF37) for highlights

**Example:**
- Icon: Warehouse icon (teal)
- Headline: "Quản lý kho hàng đa chi nhánh"
- Benefit: "Xem tất cả chi nhánh từ một nơi. Quy trình phê duyệt đơn giản. Theo dõi thời gian thực."

## Design Elements

### Icons

**Style:**
- Simple, outlined or filled
- Consistent stroke width
- Teal color (#00A8A8) or Deep Navy (#1A2B4A)

**Common Icons:**
- Warehouse (inventory)
- Chart (reports)
- Smartphone (offline)
- Gift (promotions)
- Users (multi-location)
- Checkmark (benefits)

### Illustrations

**Style:**
- Simple, professional
- Brand colors
- Business context (barbershop/salon)
- Vietnamese business setting

### Screenshots

**Guidelines:**
- Use actual product screenshots (when available)
- Anonymize real data
- Highlight key features
- Add callouts/annotations
- Vietnamese interface

### Typography

**Headlines:**
- Font: Inter Bold (700)
- Size: 28-40px (mobile), 36-48px (desktop)
- Color: Deep Navy or White

**Body Text:**
- Font: Inter Regular (400)
- Size: 16-18px
- Color: Secondary Text or White
- Line Height: 1.6

**CTAs:**
- Font: Inter Semi Bold (600)
- Size: 16-18px
- Color: White (on colored button)

## Brand Application

### Logo Usage

**Placement:**
- Bottom right corner (small)
- Top left (header, if needed)
- Watermark (subtle, if needed)

**Size:**
- Minimum: 40px height
- Maximum: 120px height (depending on layout)

### Color Application

**Primary Backgrounds:**
- Deep Navy (#1A2B4A) - Professional, trustworthy
- White (#FFFFFF) - Clean, simple
- Light Gray (#F5F5F5) - Subtle, neutral

**Accent Colors:**
- Teal (#00A8A8) - CTAs, highlights, success
- Gold (#D4AF37) - Premium, value, special offers

**Avoid:**
- Too many colors in one design
- Low contrast combinations
- Colors outside brand palette

## Platform-Specific Guidelines

### Facebook

**Best Practices:**
- Square format (1200 × 1200px) performs best
- Text overlay on images (keep text minimal)
- Clear CTA
- Mobile-optimized

**Text Overlay:**
- Keep text to 20% of image area (Facebook recommendation)
- Use high contrast
- Test readability on mobile

### Zalo

**Best Practices:**
- Similar to Facebook
- More personal tone acceptable
- Vietnamese language essential
- Visual-first approach

### Instagram

**Best Practices:**
- High-quality images
- Consistent visual style
- Use Instagram Stories for engagement
- Hashtags in caption (not on image)

## File Delivery

### Formats

**Source Files:**
- Adobe Illustrator (.ai) or Figma (.fig)
- Layered, organized
- Export-ready

**Final Files:**
- PNG (with transparency, if needed)
- JPG (for photos)
- Optimized for web (< 1MB)

### Naming Convention

**Format:** `[platform]-[type]-[topic]-[variant]-[date].[ext]`

**Examples:**
- `facebook-feature-offline-v1-2025-12-09.png`
- `instagram-value-proven-v2-2025-12-09.png`
- `zalo-cta-free-trial-v1-2025-12-09.png`

## Quality Checklist

Before finalizing any creative:

- [ ] Follows brand color palette
- [ ] Uses Inter font (or approved alternative)
- [ ] Vietnamese text properly rendered
- [ ] High contrast for readability
- [ ] Mobile-friendly (test on device)
- [ ] Correct dimensions for platform
- [ ] File size optimized (< 1MB)
- [ ] Brand logo included (if required)
- [ ] CTA clear and visible
- [ ] No typos or errors
