# Out of Scope: Billing

**Last Updated:** 2025-12-09  
**Version:** 1.0

## Overview

This document explicitly defines what is NOT included in the Billing epic (Epic 2). These items may be considered for future epics or phases.

## Explicitly Out of Scope

### Advanced Payment Features

**Not Included:**
- Payment gateway integration (MoMo, ZaloPay, etc.) - Phase 2
- Split payments (multiple customers, partial payments)
- Payment plans/installments
- Refund processing (future epic)
- Credit/debit card processing (Phase 2)
- Digital wallet integration (Phase 2)

**Rationale:** Focus on core cash and basic payment processing first. Advanced payment methods can be added in Phase 2.

### Invoice Features

**Not Included:**
- Invoice generation (PDF)
- Email invoices to customers
- Invoice numbering system (beyond basic bill number)
- Tax invoice generation
- Multi-currency invoices

**Rationale:** Basic bills are sufficient for MVP. Invoice features can be added later.

### Advanced Bill Features

**Not Included:**
- Bill templates
- Recurring bills
- Bill scheduling
- Bill estimates/quotes
- Bill approval workflows

**Rationale:** Focus on simple, fast bill creation. Advanced features not needed for MVP.

### Reporting Features

**Not Included:**
- Advanced sales reports
- Revenue analytics
- Payment method analytics
- Customer purchase history analysis

**Rationale:** Basic bill viewing is included. Advanced reporting is in Epic 7 (Reporting & Analytics).

## Future Considerations

### Phase 2 Features

- Split payments

### Phase 3 Features
