# Brand Guidelines

**Last Updated:** 2025-12-09  
**Version:** 1.0

## Overview

This directory contains comprehensive brand guidelines for the Barbershop/Beauty Chain Management System (SaaS). These guidelines ensure consistent brand identity, visual design, and user experience across all touchpoints—from web applications to marketing materials.

## Brand Foundation

### Brand Positioning

**Primary Positioning:** "Proven Multi-Location Solution"

- Battle-tested by Vietnam's largest barbershop chain (18 locations, 3 years)
- Comprehensive features that actually work
- Built for Vietnamese businesses, by people who understand Vietnamese business

### Brand Values

1. **Proven Excellence** - Features tested in real operations
2. **Local Understanding** - Deep knowledge of Vietnamese market
3. **Operational Efficiency** - Save time, reduce errors, increase control
4. **Reliable Partnership** - Partner in business growth
5. **Continuous Improvement** - Evolved based on real feedback

## Usage Guidelines

### When to Use These Guidelines

**Always use brand guidelines for:**
- Web application UI design
- Mobile application design
- Marketing website
- Marketing materials (brochures, ads, social media)
- Email communications
- Documentation and help content
- Presentations and pitch decks

### Who Should Use These Guidelines

- **UI/UX Designers** - For all interface design work
- **Graphics Designers** - For marketing materials and visual assets
- **Developers** - For implementing UI components and interactions
- **Marketing Team** - For all marketing communications and materials
- **Content Writers** - For all written content and messaging

### Getting Started

1. **Read Brand Story** - Understand the brand narrative and positioning
2. **Review Tone of Voice** - Learn how to communicate in brand voice
3. **Study Color Palette** - Use approved colors consistently
4. **Follow Typography** - Use the typography system for all text
5. **Apply UI/UX Mood & Tone** - Ensure interfaces feel right
6. **Reference Motion Principles** - Add appropriate animations

## Brand Consistency

### Do's

✅ Use approved colors from the color palette  
✅ Follow typography scale and hierarchy  
✅ Communicate in brand voice and tone  
✅ Apply UI/UX mood consistently  
✅ Use motion purposefully and professionally  
✅ Respect Vietnamese business context  
✅ Maintain accessibility standards (WCAG 2.1 AA minimum)

### Don'ts

❌ Use colors outside the approved palette  
❌ Deviate from typography system  
❌ Use off-brand voice or tone  
❌ Create inconsistent UI experiences  
❌ Add decorative animations without purpose  
❌ Ignore cultural considerations  
❌ Compromise on accessibility
