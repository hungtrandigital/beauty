# Shared Resources

## Purpose

This section contains reusable templates and assets that can be used across the project. These resources help maintain consistency and speed up documentation creation.

## Navigation

### Directories

- **[templates/](templates/)** - Reusable document templates (e.g., report-template.md, specs-template.md)
- **[assets/](assets/)** - Images, diagrams, and other media files

## How to Use Templates

1. **Browse Templates** - Check available templates in [templates/](templates/)
2. **Copy Template** - Copy the template file to your target location
3. **Customize** - Fill in the template with your project-specific information
4. **Update Links** - Update any links in the template to match your project structure
