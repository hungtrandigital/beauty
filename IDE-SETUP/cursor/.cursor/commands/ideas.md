# ideas

Use **Ideas Mode** for research, validation, and early-stage analysis.

## Steps
1) Read `INDEX.md` to understand structure
2) Follow `0-agents/mode/ideas.md`
3) If doing research, use `0-agents/agents/market-research.md`
4) Document outputs in `1-ideas/` (reports, summaries, business cases)
5) Update risks in `8-governance/risk-register.md` if found

## Output locations
- `1-ideas/1.1-market-research/` (reports, summaries, resources)
- `1-ideas/business-case-[feature].md`
- `1-ideas/1.2-initial-financing-plan.md`
- `1-ideas/1.3-initial-go-to-market-plan.md`

## Handoff
Use the orchestration format in `0-agents/mode/ideas.md` (`Current mode: ideas`, key findings, next agent).
