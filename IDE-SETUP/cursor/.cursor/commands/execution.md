# execution

Use **Execution Mode** to create strategic and creative deliverables (designs, marketing assets, content).

## Steps
1) Read `INDEX.md` for structure
2) Read `shared/context/current-scope.md` for project context
3) Read relevant plans from Plan Mode (brand guidelines, marketing strategy, requirements)
4) Follow `0-agents/mode/execution.md`
5) For designs: follow `0-agents/agents/ui-ux-designer.md` or `0-agents/agents/graphics-designer.md`
6) For marketing: follow `0-agents/agents/marketing-expert.md`
7) Create deliverables based on approved plans and brand guidelines

## Output locations
- `shared/assets/ui-ux/` (UI/UX designs, wireframes, prototypes)
- `shared/assets/graphics/` (illustrations, icons, visual elements)
- `shared/assets/brand-guidelines/visual-identity/` (brand assets)
- `4-marketing/copy/` (marketing copy, landing pages, ads, social content)
- `4-marketing/creatives/` (marketing creatives, ad visuals, campaign materials)
- `4-marketing/content/blog/` (blog posts, articles)
- `5-financing/pitches/visuals/` (pitch deck visuals)
- `shared/assets/moodboards/` (moodboards)
- `shared/assets/ui-ux/art-direction/` (art direction briefs)
- `3-technical/3.2-implementation/status/progress.md`
- `8-governance/changelog.md`

## Handoff
Use the orchestration format in `0-agents/mode/execution.md` (`Current mode: execution`, deliverables, brand alignment, Creative Director approval status, next agent).
