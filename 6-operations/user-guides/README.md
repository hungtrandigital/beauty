# User Guides

**Last Updated:** 2025-12-09  
**Version:** 1.0

## Overview

This directory contains comprehensive user guides for the Barbershop/Beauty Chain Management System. The guides are provided as static web pages with bilingual support (Vietnamese and English).

## Contents

### Main Pages

- **index.html** - Landing page with links to all user guides
- **admin.html** - Complete guide for Administrators
- **cashier.html** - Guide for Cashiers (Thu ngân)
- **warehouse-manager.html** - Guide for Warehouse Managers (Thủ kho)

### Assets

- **styles.css** - Shared stylesheet for all guide pages

## Features

- **Bilingual Support:** All guides available in Vietnamese and English
- **Language Toggle:** Easy switching between languages on each page
- **Responsive Design:** Works on desktop, tablet, and mobile devices
- **Print-Friendly:** Optimized for printing
- **Step-by-Step Instructions:** Clear, numbered steps for each task
- **Visual Indicators:** Info boxes, warnings, and success tips

## Usage

### Viewing Locally

1. Open `index.html` in a web browser
2. Navigate to the desired user guide
3. Use the language toggle buttons to switch between Vietnamese and English

### Deploying to Web Server

1. Upload all files to your web server
2. Ensure all files maintain their relative paths
3. Access via web browser at your server URL

### Updating Guides

1. Edit the HTML files directly
2. Maintain the bilingual structure (Vietnamese and English sections)
3. Update the "Last Updated" date in this README
4. Test language switching functionality

## Guide Structure

Each user guide follows this structure:

1. **Introduction** - Overview of the role and responsibilities
2. **Core Features** - Step-by-step instructions for main tasks
3. **Advanced Features** - Additional functionality
4. **Tips and Notes** - Best practices and important reminders

## Target Users

### Admin
- Full system access
- User management
- System configuration
- Reports and analytics
- Approval workflows

### Cashier (Thu ngân)
- Bill creation and management
- Payment processing
- Customer management
- Offline bill creation
- QR code scanning

### Warehouse Manager (Thủ kho)
- Inventory import/export
- Stock tracking
- Approval requests
- Inventory history
- Low stock alerts

## Maintenance
