Introduction

As a Warehouse Manager, you are responsible for managing inventory, import/export operations, and stock tracking. This guide will help you use the inventory management system efficiently.

1. Import Products to Central Warehouse

1.1. Create Import Request

  1. Go to "Inventory Management" or "Inventory" menu
  2. Select "Import"
  3. Click "Create Import Request"
  4. Select product type:
    • Regular Products: Products sold to customers
    • Dye Products: Products used for services
  5. Add products to request:
    • Select products from catalog
    • Enter import quantity
    • Add notes if needed (e.g., batch number, production date)
  6. Review request information
  7. Click "Submit Request"
  8. Request will be sent to Admin for approval
Note: Import requests must be approved by Admin before inventory is updated. You can track approval status in the request list.

1.2. View Import Requests

  1. Go to "Import Requests"
  2. View list of requests:
    • Pending: Requests awaiting Admin approval
    • Approved: Requests approved, inventory updated
    • Rejected: Requests rejected, you can view reason
  3. View details of each request

1.3. Cancel Import Request

  1. Can only cancel pending requests
  2. Select request to cancel
  3. Click "Cancel Request" and confirm

2. Export Products from Central Warehouse to Branches

2.1. Create Export Request

  1. Go to "Inventory Management" menu
  2. Select "Export"
  3. Click "Create Export Request"
  4. Select receiving branch
  5. Select product type (regular products or dye)
  6. Add products to request:
    • Select products from central warehouse inventory
    • Enter export quantity
    • System automatically checks available quantity
  7. Review request information
  8. Click "Submit Request"
Warning: You cannot export more than available quantity in central warehouse. System will warn if quantity is insufficient.

2.2. Approval and Confirmation Process

  1. After submitting request, Admin will review and approve
  2. When approved, branch will receive notification
  3. Branch must confirm receipt
  4. After branch confirmation, inventory is updated:
    • Central warehouse: Decrease quantity
    • Branch: Increase quantity

2.3. View Export History

  1. Go to "Export History"
  2. View all export requests made
  3. Filter by branch, date, or status
  4. View details of each request

3. Return Products from Branch to Central Warehouse

3.1. Create Return Request

  1. Go to "Inventory Management" menu
  2. Select "Return"
  3. Click "Create Return Request"
  4. Select returning branch
  5. Select products from branch inventory
  6. Enter return quantity
  7. Add notes (e.g., excess inventory, expired products)
  8. Submit request

3.2. Approval Process

Similar to export, return requests need Admin approval and confirmation before inventory is updated.

4. View Inventory

4.1. View Central Warehouse Inventory

  1. Go to "Inventory"
  2. Select "Central Warehouse"
  3. View list of all products and stock quantities
  4. Use search to find specific products
  5. Filter by product type (regular products/dye)

4.2. View Branch Inventory

  1. Go to "Inventory"
  2. Select "Branch" and select specific branch
  3. View that branch's inventory

4.3. View Consolidated Inventory

  1. Select "All Locations"
  2. View total inventory across all locations
  3. View quantity at each location

4.4. Low Stock Alerts

System automatically displays alerts when inventory is low:

  • Products with stock below threshold will be marked in red
  • You can view list of all products needing restock
  • Report to Admin to decide on import

5. Inventory History

5.1. View Import/Export History

  1. Go to "Inventory History"
  2. View all activities:
    • Imports to central warehouse
    • Exports from central warehouse
    • Returns from branches
    • Inventory adjustments (if you have permission)
  3. Filter by:
    • Activity type
    • Date
    • Product
    • Branch

6. Inventory Adjustment (If You Have Permission)

6.1. When Adjustment is Needed

Inventory adjustment is used when:

  • Discrepancy found between actual and system inventory
  • Damaged or lost products
  • Periodic inventory count

6.2. Create Adjustment Request

  1. Go to "Inventory Adjustment"
  2. Select location (central warehouse or branch)
  3. Select product to adjust
  4. Enter adjustment quantity:
    • Increase: Enter positive number (e.g., +10)
    • Decrease: Enter negative number (e.g., -5)
  5. Enter adjustment reason (required)
  6. Submit request (needs Admin approval)
Important: Inventory adjustments must have clear reason and Admin approval. All adjustments are logged in history for audit.

7. Product and Dye Management

7.1. Distinguish Products and Dye

The system separates two types:

  • Regular Products: Products sold to customers (shampoo, cream, etc.)
  • Dye Products: Products used for services (hair dye, chemicals, etc.)

Each type has separate inventory and is managed independently.

7.2. View Product Information

  1. In inventory list, click on product name
  2. View detailed information:
    • Product name and description
    • Selling price
    • Current inventory at each location
    • Import/export history
