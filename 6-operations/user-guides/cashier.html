Introduction

As a Cashier, you are responsible for creating bills, processing payments, and managing customers at the branch. This guide will help you use the system efficiently.

1. Create New Bill

1.1. Create Bill for Customer

  1. Go to "Bills" menu
  2. Click "New Bill" button
  3. Add customer (if applicable):
    • Scan customer QR code to automatically load information
    • Or search customer by name, phone number
    • Or quickly create new customer
  4. Add products to bill:
    • Select products from list
    • Enter quantity
    • System automatically calculates price (branch price or default price)
  5. Add services to bill:
    • Select services from list
    • System automatically calculates price
  6. View total amount and applied promotions
  7. Click "Save Draft" if not paid yet, or "Pay" to process payment immediately
Note: The system automatically applies eligible promotions for customers. You can view applied promotions in the bill summary section.

1.2. Edit Bill

  1. Find the bill to edit (not yet paid)
  2. Click on the bill to open
  3. Add, edit, or remove products/services
  4. System automatically recalculates total
  5. Save changes

1.3. Delete Bill

  1. Can only delete unpaid bills
  2. Click "Delete" button and confirm

2. Create Bill Offline

2.1. When Internet Connection is Lost

The system supports creating bills when there's no internet:

  1. When connection is lost, system automatically switches to offline mode
  2. You can still create bills normally
  3. Bills are saved locally on the device
  4. When connection is restored, system automatically syncs bills to server

2.2. Check Sync Status

  1. Go to "Pending Sync Bills" menu
  2. View list of bills not yet synced
  3. System will automatically sync when connection is available
  4. If there are errors, you'll see notification and can retry sync
Warning: In offline mode, system uses previously cached inventory data. When syncing, system will recheck actual inventory.

3. Process Payment

3.1. Cash Payment

  1. Select bill to pay
  2. Click "Pay"
  3. Select payment method: "Cash"
  4. Enter amount received from customer
  5. System automatically calculates change
  6. Click "Confirm Payment"
  7. Print receipt (if needed)

3.2. Card Payment

  1. Select bill to pay
  2. Click "Pay"
  3. Select payment method: "Card"
  4. Enter amount (usually equals bill total)
  5. Click "Confirm Payment"
  6. Print receipt

3.3. Mixed Payment

You can split bill into multiple payment methods:

  1. Select bill
  2. Click "Pay"
  3. Add first payment method (e.g., cash 500,000 VND)
  4. Click "Add Payment Method" and add second method (e.g., card 200,000 VND)
  5. Check total paid amount equals bill total
  6. Click "Confirm Payment"

3.4. Partial Payment

If customer pays partially:

  1. Enter amount customer pays (less than total)
  2. System will mark bill as "Partially Paid"
  3. You can pay remaining amount later

4. Scan Customer QR Code

4.1. Using QR Scan

  1. When creating new bill, click "Scan QR" button
  2. Allow system to access camera
  3. Point camera at customer's QR code
  4. System automatically recognizes and loads customer information:
    • Customer name
    • Phone number
    • Current loyalty points
    • Membership tier
    • Active promotions
  5. Customer information is automatically filled into bill
Tip: Scanning QR helps you quickly identify customers and apply appropriate benefits, saving time.

5. Customer Management

5.1. Search Customer

  1. Go to "Customers" menu
  2. Use search bar to find by:
    • Customer name
    • Phone number
    • Email
  3. View customer detailed information

5.2. Create New Customer

  1. Click "Add Customer"
  2. Fill in basic information:
    • Name (required)
    • Phone number (required)
    • Email (optional)
  3. System automatically generates QR code for customer
  4. Save information

5.3. View Transaction History

  1. Select customer from list
  2. Go to "History" tab
  3. View all customer bills
  4. View loyalty points and membership tier

6. View Bill History

6.1. Search Bills

  1. Go to "Bill History" menu
  2. Search by:
    • Bill number
    • Customer name
    • Date
    • Branch

6.2. View Bill Details

  1. Select bill from list
  2. View details:
    • Customer information
    • Product/service list
    • Item prices
    • Applied promotions
    • Total amount
    • Payment method

6.3. Reprint Receipt

  1. Select bill to print
  2. Click "Print Receipt" or "Print" button
  3. Confirm print

7. Promotions and Discounts

7.1. Automatic Promotions

System automatically applies promotions when:

  • Customer is eligible
  • Bill meets conditions (e.g., total over 500,000 VND)
  • Promotion is within application period

You can view applied promotions in the bill summary section.

7.2. Manual Discount

If manual discount is needed (with appropriate permissions):

  1. In bill, click "Manual Discount"
  2. Enter discount amount or percentage
  3. Enter discount reason (if required)
  4. Apply discount

7.3. Apply Voucher

  1. If customer has voucher, click "Apply Voucher"
  2. Enter voucher code
  3. System checks and applies voucher if valid
  4. View voucher discount in summary

8. Tips and Notes

üí° Usage Tips

  • Use QR scan to save time when creating bills
  • Save draft bills if customer hasn't decided yet
  • Check automatic promotions before applying manual discount
  • Print receipt immediately after payment to avoid forgetting

‚ö†Ô∏è Important Notes

  • Cannot edit bills after payment
  • Double-check amount before confirming payment
